# 超市管理系统 java847

## 项目概述

本项目是一款实现商品进销存的超市管理系统。基于前后端分离的架构设计，后端采用 JavaWeb 技术栈，前端使用 Vue.js 进行构建。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

- 商品管理
  - 商品信息维护
  - 商品分类管理
- 进货管理
  - 进货单据处理
  - 供应商信息管理
- 销售管理
  - 销售单据处理
  - 客户信息管理
- 库存管理
  - 库存查询
  - 库存预警
- 系统管理
  - 用户权限管理
  - 系统日志记录

## 系统角色

- 系统管理员
- 商品管理人员
- 采购人员
- 销售人员

## 运行环境

- 后端
  - JDK 1.8 或以上版本
  - MySQL 5.7 或以上版本
- 前端
  - 现代浏览器

## 部署步骤

1. 准备所需环境（JDK, MySQL）
2. 初始化数据库，执行数据库脚本
3. 部署后端服务
4. 部署前端应用

## 目录结构

